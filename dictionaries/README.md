# Словари

Каждый словарь — обычный текстовый файл `.txt`: **одно слово на строку**, кодировка UTF-8.

## В репозитории

- `example.txt` — небольшой пример для проверки бота.

Колоды из коммерческих изданий настольной игры Alias **не включены** в репозиторий из‑за возможных ограничений авторского права. Добавьте свои файлы локально или через `/dict_upload` (нужен `ADMIN_IDS` в `.env`).

## Свой словарь

1. Создайте `my-words.txt` в этой папке или загрузите через бота.
2. Перезапуск не обязателен: новые `.txt` в `dictionaries/` подхватываются при выборе словаря.
