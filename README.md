# Alias Bot for Telegram

A simple and convenient bot for playing "Alias" with friends directly in Telegram.

## üöÄ Features

  * **üé≤ Random Words:** Get words from the active dictionary with a single button press.
  * **üîó Precise Definitions:** Each word is a clickable link to **Wiktionary** for a quick and accurate definition.
  * **‚öôÔ∏è Flexible Settings:** A convenient menu to change the active dictionary and interface language.
  * **üåç Bilingual:** The bot fully supports Russian and English.
  * **‚úèÔ∏è Dictionary Management:**
      * Upload your own dictionaries with the `/dict_upload` command.
      * Add new words to any dictionary with the `/addword` command.
  * **üíæ Session Persistence:** The bot remembers your chosen language and dictionary even after a restart.

## üîß Installation and Launch

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/renkagod/Alias.git
    cd Alias
    ```

2.  **Install dependencies:**

    ```bash
    pip install -r requirements.txt
    ```

3.  **Configure environment variables:**

      * Create a `.env` file in the root folder.
      * Add your token to it: `BOT_TOKEN="YOUR_TOKEN_FROM_BOTFATHER"`

4.  **Start the bot:**

    ```bash
    python bot.py
    ```

## ü§ñ Commands

  * `/start` - Start the bot and complete the initial setup.
  * `/dict_upload` - Upload your own `.txt` file with words.
  * `/addword` - Add one or more new words to an existing dictionary.
